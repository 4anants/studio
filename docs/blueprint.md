# **App Name**: FileSafe

## Core Features:

- Employee Login: Secure login for employees to access the document repository.
- Admin Login: Secure login for administrators with elevated privileges.
- Document Upload: Functionality for users and admins to upload documents (salary slips, medical reports, appraisal letters, personal documents).
- Document Download: Functionality for users and admins to download uploaded documents.
- User-Specific Uploads: Employees can upload personal documents specifically for HR/Admin use.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to inspire trust and security.
- Background color: Light blue (#E8EAF6) for a calm and professional atmosphere.
- Accent color: Teal (#009688) to highlight important actions and elements.
- Body and headline font: 'Inter', a sans-serif font, to promote a clean, neutral and modern user experience.
- Use clear and intuitive icons for file types (PDF, DOC, etc.) and actions (upload, download).
- Maintain a clean and organized layout to allow effortless file navigation.
- Use subtle animations during file upload and download processes to provide feedback to the user.
